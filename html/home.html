Important FAQ

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is like a map of a web page. It turns the page into a tree of objects that JavaScript can use to change how the page looks and works. This (DOM) make web pages interactive and dynamic.

2. How do you select an element from the DOM?

Selecting an element from the DOM (Document Object Model) can be done using many methods in JavaScript.here are the some ways

  • 1.By ID
  • 2.By Class Name
  • 3.By Tag Name
  • 4.By CSS Selector
    *Single Element by querySelector it give us fast element
    *Multiple Elements querySelectorAll it give us all element of this class
  • 5.By Name Attribute

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where I add a single event listener to a parent element instead of multiple listeners to each child element. the parent element then handles events for all its children.
It’s Useful because Fewer Listeners
Handles New Elements
Cleaner Code

4. How do you manipulate an element's attributes and styles using the DOM?

Manipulating an element’s attributes and styles using the DOM can be done in many ways. Here are some methods:
Get Attribute: const element = document.getElementById("myElement"); const value = element.getAttribute("attributeName");
Set Attribute: element.setAttribute("attributeName", "AttributeNewValue");
Remove Attribute element.removeAttribute("attributeName");
