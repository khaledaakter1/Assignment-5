1. What is the Document Object Model (DOM)?

2. How do you select an element from the DOM?

You can select an element from the DOM using methods like getElementById(), querySelector(), or getElementsByClassName().

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is attaching a single event listener to a parent element to handle events from its child elements, improving performance.

4. How do you manipulate an element's attributes and styles using the DOM?

You manipulate attributes with methods like setAttribute() and access styles via element.style.property or classList methods for classes.
