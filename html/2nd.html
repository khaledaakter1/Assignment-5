1. What is the Document Object Model (DOM)?

The DOM is a structured representation of a webpage's elements.

2. How do you select an element from the DOM?

To select an element from the DOM using methods like getElementById(), querySelector(), or getElementsByClassName() to target specific elements.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation allows handling events at a parent element, improving performance and managing dynamic elements.

4. How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes using setAttribute(), getAttribute(), and modify styles directly via element.style or classList.
